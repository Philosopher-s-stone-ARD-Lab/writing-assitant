# Contributing to Writer's Companion

**Important Notice**: This application was created by Islam Salahuddin for his personal writing project "أسفار الوحدة والتكوين" (The Books of Loneliness and Genesis). While the code is shared as open source to demonstrate custom LLM applications for creative writing, the repository is not intended for community contributions or modifications. However, if you have any ideas to improve the application, please feel free to reach out to the author, or submit a pull request directly.

## Project Purpose

This repository is shared as an example and proof-of-concept for:
- Custom LLM applications for creative writing
- Specialized AI personas for narrative assistance
- Integration of AI tools in specific writing workflows

## Usage Guidelines

You may:
- Study the code to understand custom LLM application development
- Use concepts and patterns for your own writing projects
- Reference this project in academic or technical discussions
- Submit pull requests for modifications

Please do not:
- Use this as a general-purpose writing tool
